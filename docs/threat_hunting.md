# 🎯 Threat Hunting
>BlueTeam 
- SOC‑N5

La chasse aux menaces est proactive et orientée hypothèses.

# 🧠 Méthodologie

1. Définir une hypothèse
2. Collecter les données
3. Exécuter les queries
4. Analyser les résultats
5. Documenter
6. Améliorer la détection
