# 📘 Documentation
>BlueTeam
- SOC‑N5

Ce dossier contient **l’ensemble de la documentation stratégique, tactique et opérationnelle** du projet BlueTeam, conçu pour un **SOC militaire de niveau national (SOC‑N5)**.

La documentation couvre :

- L’architecture SOC  
- L’ingénierie de détection  
- La réponse aux incidents  
- La chasse aux menaces  
- Les playbooks SOC  
- La préparation opérationnelle  
- Les pipelines SIEM  
- Les normes et méthodologies

# 🛰️ Vision Globale

La documentation BlueTeam est conçue pour :

- Standardiser les opérations SOC  
- Améliorer la détection et la visibilité  
- Accélérer la réponse aux incidents  
- Structurer la chasse aux menaces  
- Renforcer la posture cyber défensive  
- Maintenir un état de **combat cyber permanent**
